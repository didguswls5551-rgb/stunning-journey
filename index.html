기록

관리자

로그인 코드 관리 관리자 코드로 접속한 사람만 로그인 코드를 추가·삭제할 수 있습니다. 코드 목록은 Firebase Firestore에 저장되어 모든 기기에서 공유됩니다.
Firebase 연결 상태를 확인하는 중입니다.

의뢰 내용

사용 안내

1. 로그인 코드 입력 발급받은 로그인 코드를 입력한 뒤 로그인 버튼을 누르면 Note 작성 화면으로 이동합니다.
2. Note 작성 의뢰자, 촬영일, 제목, 촬영 내용, 촬영장소, 기자 정보를 직접 입력할 수 있습니다.
3. 의뢰 내용 자동 입력 Note 화면의 의뢰 내용 버튼을 누른 뒤 의뢰서를 붙여넣고 입력을 누르면 주요 항목이 자동으로 정리됩니다.
4. 저장 저장 버튼을 누르면 현재 로그인 코드 기준으로 내용이 저장됩니다.
